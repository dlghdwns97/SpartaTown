# SpartaTown
게더 클론 스파르타 타운 만들기

## 💻 프로젝트 소개
게더를 참고해 만든 온라인 접속 플랫폼입니다.

## 📆 프로젝트 개발 기간
2023-09-04 ~ 2023-09-07

### 🏗️ 개발 환경
- `Unity`
- `C#`

## 🛠️ 주요 기능
#### 캐릭터 이동 및 마우스 방향 좌우 바라보기
#### 캐릭터를 따라가는 카메라와 이름
#### 캐릭터 애니메이션
#### 접속 시 이름 입력, 캐릭터 변경 가능
#### 참석 인원을 UI로 확인 가능
#### 인게임에서 이름 변경, 캐릭터 변경 가능
#### 실시간 컴퓨터 시간 표시
